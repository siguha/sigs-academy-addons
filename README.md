
Sig's Academy Addons
=======
A **QoL mod packaged with a variety of features**, built specifically for the Cobblemon Academy: Season 2 Minecraft server.

As most of the components of this mod target features exclusive to this server, this mod will be useless to anyone trying to use it outside of it.

### A Note Before Use
This is an **unofficial**, **unverified** mod. These features aren't in anyway classified or designed to be cheats or to give those who use it a one-up over those who dont. It's simply meant to make the lives of the players a bit easier. That being said, this is a __**use at your own risk**__ mod as I don't want to really go through the process of waiting three weeks for a response back on whether or not it's OK.

Contributions
============
PRs are welcome to help better the community experience. If you're interested, please feel free to help expand QoL features to other aspects of the server that are otherwise irritating to play/navigate.

Don't want to contribute code? Open up a issue and specify the feature you'd like added.

Issues
============
Please ensure your issues have at least:
* Mod version, Academy Pack version
* Description of issue
* Reproduction steps
* Log/crash report (`logs/latest.log` or `crash-reports/(latest crash report by date)`)
* Additional installed mods *outside of the base Academy pack*
* Screenshots if the problem is visual

This is a hobby project. It may not recieve frequent or consistent updates. If you want specific features, consider adding them yourselves or fork the project.
